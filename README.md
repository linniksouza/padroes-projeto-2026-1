# Padrões de Projeto de Software com Java - 2026/1

## Objetivo

Este é um repositório da disciplina de Padrões de Projetos de Software com Java, da instituição de ensino Estácio Amazonas, para o semestre 2026/1. A ideia é compartilhar com os alunos exemplos de implementação criados durante as aulas.

## Estrutura

O repositório será organizado de acordo com a estrutura de conteúdos apresentados durante as aulas. A cada aula, será adicionado um novo commit contendo todos os exemplos implementados no dia. Além disso, poderão ser criados também exemplos avulsos, de acordo com a demanda da turma.
